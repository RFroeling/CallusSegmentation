# General project folder

For each project, its own Git repository will be made, and it will be pushed to GitLab. 

- To make a repo:
```bash
git init
```

- Initialize empty remote via GitLab
- Add remote:
```bash
git remote add origin git@git.wur.nl:froel001/[remotename].git
```

- Push to remote:
```bash
git push --set-upstream origin main
```
